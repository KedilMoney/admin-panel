# admin-panel — Kedil Admin Panel

Internal admin dashboard for Kedil. Used to manage users, view data, and monitor the app.

## Stack
- **Framework:** Next.js (TypeScript)
- **Deployment:** Netlify (auto-deploys on push to main)

## Key Commands
```bash
yarn dev     # Start dev server on localhost:3000
yarn build   # Build for production
yarn lint    # Run ESLint
```

## Related Repos
- **kedil_be** — backend API this panel communicates with
